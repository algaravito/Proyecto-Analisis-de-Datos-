## 📂 Proyectos


---

### 🔹  Análisis de Cancelación de Clientes (Churn) – Model Fitness

**Descripción:**  
Proyecto enfocado en identificar patrones de abandono de clientes (churn) en una cadena de gimnasios mediante el análisis del comportamiento de los usuarios.

**Enfoque principal:**
- Identificación y análisis de churn  
- Análisis de comportamiento de clientes  
- Análisis Exploratorio de Datos (EDA)  
- Segmentación de clientes y métricas clave  
- Estrategias de retención basadas en datos  

**Herramientas:** Python, Pandas, NumPy, Matplotlib, Seaborn  
📁 Carpeta: ` Análisis de Cancelación de Clientes (Churn) – Model Fitness`

---

### 🔹 Proyecto SQL – Análisis de Servicio de Libros

**Descripción:**  
Análisis de una base de datos de un servicio de libros que incluye información sobre libros, autores, editoriales, calificaciones y reseñas de usuarios.

**Enfoque principal:**
- Escritura y optimización de consultas SQL  
- Exploración de datos en múltiples tablas  
- Análisis de autores, libros y editoriales  
- Identificación de patrones en reseñas y calificaciones  
- Generación de insights para decisiones de negocio  

**Herramientas:** SQL  
📁 Carpeta: ` Proyecto SQL – Análisis de Servicio de Libros`

---

### 🔹 Proyecto A/B Testing – Sistema de Recomendación

**Descripción:**  
Evaluación de un experimento A/B en una plataforma de e-commerce para analizar el impacto de un sistema de recomendación en el comportamiento de los usuarios.

**Enfoque principal:**
- Análisis y validación de pruebas A/B  
- Análisis de embudo (product_page → product_cart → purchase)  
- Comparación de tasas de conversión  
- Análisis estadístico de resultados  
- Evaluación de cambios en producto y experiencia de usuario  

**Herramientas:** Pandas, NumPy, Matplotlib, Statsmodels  
📁 Carpeta: `ab-testing-recommender-system`
